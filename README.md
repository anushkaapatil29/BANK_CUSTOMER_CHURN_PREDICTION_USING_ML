# BANK_CUSTOMER_CHURN_PREDICTION_USING_ML
End-to-end machine learning system predicting customer churn with 70% accuracy. Engineered feature pipeline, trained 3 models (Logistic Regression, Random Forest, XGBoost), handled class imbalance with SMOTE, deployed interactive Streamlit web app. SHAP explainability reveals churn drivers. Estimated $2.1M annual retention value.
